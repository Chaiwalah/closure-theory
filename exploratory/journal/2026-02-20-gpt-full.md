## GPT's Four Pillars of Where to Look

### 1. Fine Structure Survival Tests
If the channel eats high-frequency info, anomalies show up wherever inference assumes details are preserved.
- y₁₀ vs χ²/dof is the sharpest hook
- Should replicate across: SNLS vs DES vs PS1 vs low-z surveys
- Should replicate across fitters: SALT2 vs SALT3 vs BayeSN
- Should replicate across wavelengths: optical-only vs optical+NIR

### 2. Redshift-Dependent Model Breakdowns
Channel effect ≠ simple mean offset. It looks like: correction works until it doesn't.
- High-z residual coupling (color↔distance only at z>0.5)
- Pattern: no signal nearby → weak mid-z → strong high-z = accumulation
- Discriminator: WHICH features break and HOW (vs population evolution faking it)

### 3. Cross-Messenger Consistency
Two ways to measure same distance through different channels:
- EM vs GW standard sirens (PRIMARY test)
- Strong lensing time delays (light curve timing + mass model)
- BAO (geometry) vs SN (photometric) comparisons by sky position/foreground

### 4. Angular-Scale Fingerprint
y₁₀ hitting at ~10 arcmin needs diagnosis:
- If peak matches instrument beam/map resolution → ARTIFACT (red flag)
- If peak matches physical scale (filaments/groups/cluster outskirts) → REAL (green flag)
- Test: does signal persist across tracers/instruments at same physical scale?

### A. Quasar Absorption / Lyman-α Forest
- Spectral fine-structure → cosmology pipeline

### B. Time-Domain Sharpness Degradation

- If channel smears that → systematic bias mimicking "lensing is low"
- Elephant signature: shear calibration residuals correlating with free-electron/tSZ/dust after standard corrections
- THIS COULD EXPLAIN THE S8 TENSION DIRECTLY

### D. Separability Assumption Violations
- Modern stacks assume: source × instrument × simple_dust (separable)
- Closure says: source ⊗ channel ⊗ measurement (coupled)
